✨ Giới thiệu về GitHub

Tớ giới thiệu cho cậu về GitHub, một trong những trang web quan trọng nhất dành cho lập trình viên, nhà phát triển, và học sinh đam mê công nghệ 😎

🧠 GitHub là gì?

GitHub là một nền tảng lưu trữ mã nguồn và hợp tác lập trình dựa trên Git (một hệ thống quản lý phiên bản). Nó cho phép:

  • Lưu trữ và chia sẻ code (dự án lập trình)
  • Làm việc nhóm (nhiều người cùng sửa code)
  • Theo dõi lịch sử thay đổi trong code
  • Tạo trang web miễn phí (GitHub Pages)

🚀 Các tính năng chính:

1️⃣ 🗂️ Lưu trữ code miễn phí:

Tạo repository (dự án) để lưu code, hình ảnh, tài liệu, v.v.

2️⃣ ⏳ Theo dõi thay đổi:

Khi bạn chỉnh sửa, GitHub sẽ lưu lịch sử mỗi lần thay đổi (gọi là commit).

3️⃣ 🤝 Làm việc nhóm:

Nhiều người có thể cùng chỉnh sửa một dự án, chia nhánh code (branch), sửa lỗi, thêm tính năng.

4️⃣ 🖥️ Tạo trang web miễn phí với GitHub Pages:

Các bạn có thể làm một website HTML/CSS và đăng lên mạng miễn phí với địa chỉ như:

📍 https://ten-nguoi-dung.github.io/ten-du-an

5️⃣ 🧪 Quản lý dự án:

  • Tạo ghi chú công việc (issues)
  • Quản lý tiến độ học hay viết phần mềm
  • Giao tiếp với bạn cùng nhóm

💡 Ví dụ sử dụng:

  • Các bạn viết một website đơn giản bằng HTML và CSS → tải lên GitHub → chia sẻ link cho bạn bè.
  • Các bạn viết phần mềm → lưu code trên GitHub → người khác cũng có thể xem và đóng góp.
